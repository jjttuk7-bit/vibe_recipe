- **①추출**: `수정/2-01_고추장포뮬러.png` → 고추장2T:설탕1T:다진마늘1t:식초1T:참기름1T:깨1t. flavor=매콤·달콤·새콤·고소. tip=아삭하려면 오이 절여 5분 후 물기 닦고. `lib/formulas.ts` ratios 에 verified:false 로 추가, 배포.
- **②테스트(2026-06-29)**: 유케이가 오이무침 빌드 → **비율은 정확**(고추장2T·설탕1T·식초1T·마늘1t·참기름1T 일치 ✅). **그러나 셰프가 간장 1T를 임의 추가** ❌ — 고추장 무침 공식엔 간장 없음. 일반 무침 지식 환각.
- **③수정(2026-06-29)**: `renderKnowledgePack` 규율 강화 — "공식 적용 시 그 공식 재료 구성만, 공식에 없는 양념(예: 간장) 임의 추가 금지". 재배포.
- **④확인(대기)**: 재테스트에서 간장 안 들어가면 → 유케이가 비율 확인 → verified:true.
- **배운 것**: 비율 추출은 잘 됨. 문제는 *재료 집합 준수* — mini가 공식 외 재료를 끼움. 규율로 1차 차단, 안 되면 재료 집합을 구조화(명시 리스트)로 강제 검토.
